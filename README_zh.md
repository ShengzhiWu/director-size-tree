# Director Size Tree

一个用于查看磁盘和文件夹占用空间的桌面小工具。

它会把磁盘、文件夹和较大的文件画成按尺寸缩放的方块。方块越高，代表占用空间越大。你可以用它快速判断空间主要被哪些目录或文件占用了。

## 功能

- 查看整台计算机各个磁盘的空间占用。
- 输入一个文件夹路径，只扫描这个文件夹。
- 扫描过程中渐进显示结果，不必等全部扫描完成。
- 鼠标悬停时高亮当前块及其父级路径。
- 双击文件夹块可在系统文件管理器中打开对应文件夹。
- 可保存扫描结果为 JSON，之后重新载入，避免重复等待长时间扫描。
- 可切换是否在块上显示名称。

## 使用

安装依赖：

```bash
npm install
```

启动应用：

```bash
npm start
```

打开后：

1. 直接点击 `Scan`：扫描整台计算机。
2. 在输入框中填写文件夹路径后点击 `Scan`：只扫描该文件夹。
3. 使用 `File -> Save Results...` 保存结果。
4. 使用 `File -> Load Results...` 载入之前保存的结果。

## 说明

扫描大目录可能需要较长时间。这是正常现象，因为系统通常不会提前保存每个文件夹的总大小，应用需要实际遍历其中的文件和子文件夹。
